# RealtimeChat

## Install Dependancies
<ul>
  <li>npm install</li>
  <li>cd client</li>
  <li>npm install</li>
</ul>

## Setup Project
<ul>
  <li>Create a config folder in root directory</li>
  <li>Create a env.json file in config folder</li>
  {
    "MONGO_URI": mongodb connection link, 
    "JWT_SECRET": "some secret string"
  }
</ul>

## Run Project
<ul>
  Server -
  <li>npm run server</li>
  <br />
  Client -
  <li>cd client</li>
  <li>npm start</li>
</ul>
